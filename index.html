SimMaster Cheats

Abrir o console de cheats

Para usar outros cheats, primeiro abra o console e habilite os cheats.

PC: Ctrl + Shift + C
Mac: Command + Shift + C
PlayStation: L1 + L2 + R1 + R2
Xbox: LB + LT + RB + RT

Habilitar cheats

Para usar outros cheats, primeiro você deve habilitar cheats no jogo.

testingcheats true

Adicionar Simoleons

Adiciona 1000 Simoleons à família.

kaching

Adiciona 50.000 Simoleons à família.

motherlode

Editar Sim completamente

Permite editar completamente um Sim no Criar um Sim.

cas.fulleditmode

Resetar um Sim

Reseta um Sim que está preso ou bugado.

resetSim [primeiro nome] [último nome]

Aumentar habilidades

Aumenta uma habilidade específica de um Sim.

stats.set_skill_level [nome da habilidade] [nível]

Desbloquear todas as recompensas de construção

Desbloqueia todos os itens de recompensas de carreira no modo construção.

bb.ignoregameplayunlocksentitlement

Desbloquear objetos ocultos

Desbloqueia todos os objetos ocultos no modo construção.

bb.showhiddenobjects

Mover objetos livremente

Permite mover objetos livremente sem restrições de grade no modo construção.

bb.moveobjects

Desligar e Ligar envelhecimento

Desliga o envelhecimento automático para a família ativa.

aging off

Liga o envelhecimento automático para a família ativa.

aging on

Mudar os dias de vida de um Sim

Muda os dias de vida de um Sim específico.

modifyrelationship [primeiro nome] [último nome] [primeiro nome alvo] [último nome alvo] [quantidade] Friendship_Main
modifyrelationship [primeiro nome] [último nome] [primeiro nome alvo] [último nome alvo] [quantidade] Romance_Main
